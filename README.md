A Python package for statistical analysis of CSV datasets with built-in visualization.

## Features

- **Core Statistics**:
  - Mean, median, mode
  - Variance, standard deviation
  - Correlation matrices

- **Visualizations**:
  - Histograms
  - Boxplots
  - Scatter plots

- **Formats**:
  - CSV file input
  - Terminal output
  - PNG image exports
